# My React App

This project is a simple React application built with TypeScript and Vite, utilizing Tailwind CSS for styling. The application features a landing page with links to two different forms for entering an address.

## Project Structure

```
my-react-app
├── src
│   ├── components
│   │   ├── CascadingInputForm.tsx
│   │   ├── FreeInputForm.tsx
│   │   └── LandingPage.tsx
│   ├── App.tsx
│   ├── main.tsx
│   └── styles
│       └── index.css
├── public
│   └── index.html
├── package.json
├── tsconfig.json
├── tailwind.config.js
├── postcss.config.js
└── README.md
```

## Features

- **Landing Page**: The main entry point of the application that provides links to two forms.
- **Cascading Input Form**: A form where users can enter their address with a submit button.
- **Free Input Form**: Another form for entering an address with a submit button.

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd my-react-app
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm run dev
   ```

5. Open your browser and go to `http://localhost:3000` to view the application.
